# Google Sheets 연동 설정 (신청 데이터 저장)

새출발금융센터 랜딩페이지 때와 동일한 방식입니다.

## 1. 스프레드시트 준비
1. Google Sheets에서 새 스프레드시트를 만듭니다. (예: "컬러인에듀센터 신청내역")
2. 상단 메뉴 `확장 프로그램 → Apps Script` 클릭

## 2. 스크립트 붙여넣기
1. 열린 Apps Script 편집기에서 기본 코드를 모두 지우고, 같은 폴더의 `Code.gs` 내용을 붙여넣습니다.
2. 저장 (Ctrl/Cmd + S)

## 3. 웹앱으로 배포
1. 우측 상단 `배포 → 새 배포`
2. 유형 선택에서 톱니바퀴 → `웹앱` 선택
3. 설정:
   - 설명: `컬러인에듀센터 신청폼`
   - 실행 계정: `나`
   - 액세스 권한이 있는 사용자: `모든 사용자` (브라우저에서 직접 호출하기 위해 필요)
4. `배포` 클릭 → 권한 승인 (본인 계정으로 로그인 → 고급 → 이동 허용)
5. 발급된 **웹 앱 URL**을 복사합니다. (예: `https://script.google.com/macros/s/AKfycb.../exec`)

## 4. 웹앱 URL 연결
`js/main.js` 파일 상단의 `WEBHOOK_URL` 값을 방금 발급받은 웹 앱 URL로 교체하세요.

```js
const WEBHOOK_URL = "https://script.google.com/macros/s/여기에_배포된_ID/exec";
```

## 5. 시트 구조
처음 신청이 들어오면 자동으로 `신청내역` 시트와 헤더(타임스탬프 / 서비스 / 이름 / 연락처 / 희망 일정 / 문의 내용)가 생성됩니다.

## 스크립트 수정 후 재배포
`Code.gs` 내용을 수정했다면 `배포 → 배포 관리 → 수정(연필 아이콘) → 새 버전` 으로 다시 배포해야 반영됩니다. URL은 그대로 유지됩니다.
